gifmaker
========

.. image:: docs/brain.gif

Convert a series of images into a gif animation.

Installation
------------

.. code-block::
    pip install -e .

Usage
-----

.. code-block::
    gifmaker -i IMAGE1.png IMAGE2.png
    gifmaker -i *.png

Note: all images mush have the same size.

To see all the features, run:

.. code-block::
    gifmaker -h
