Convert a series of images into a gif animation, with the possibility to apply various processing on the input image (cropping, rescaling, etc.).

InstallationÔÉÅ

pip install gifmaker

UsageÔÉÅ

The examples below are based on the example images: gifmaker/tests/images/. To look at all the features, run:

gifmaker -h

Standard usageÔÉÅ

gifmaker -i 1.png 2.png 3.png 4.png 5.png -o myanim.gif

Or, using Unix regex:

gifmaker -i *.png

Note

All images should have the same size.

CroppingÔÉÅ

gifmaker -i *.png -c 10 100 40 100 -o anim-crop.gif

RescalingÔÉÅ

For 2x downsampling:

gifmaker -i *.png -r 0.5 -o anim-ds.gif

You can combine processes. Example: cropping and 2x upsampling (in the same order):

gifmaker -i *.png -c 50 100 50 100 -r 2 -o anim-ds.gif
